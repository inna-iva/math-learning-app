# Math Learning App (Java)

A simple educational Java console application that helps users practice basic arithmetic operations.  
The app generates random math problems, tracks the user’s score, and provides performance feedback after each session.

---

## Features
- Randomly generates arithmetic problems (+, −, ×, ÷)
- Keeps score and displays results at the end of each session
- Adjustable difficulty levels (easy, medium, hard)
- Input validation for user-friendly experience
- Modular, object-oriented code design

---

## Concepts Demonstrated
- Java OOP principles (classes, objects, methods)
- Control flow and exception handling
- Random number generation
- User input and validation
- File I/O (for saving results)
